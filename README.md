# 🎓 Student Marksheet System (JavaScript)

## 📌 About This Project
This is a simple JavaScript-based **Student Marksheet System** that:
- Calculates grades 📊
- Determines pass/fail status ✅❌
- Provides remarks 📝 based on marks

It is designed for beginners to understand **core programming concepts** like conditional statements and logic building.

---

## 🎯 Objective
- Practice JavaScript fundamentals
- Learn how to use `if-else`, `switch`, and operators
- Build logical thinking for real-world problems

---

## ⚙️ Features
- Grade calculation (A, B, C, F)
- Pass/Fail result system
- Remarks generation (Excellent, Good, Average, Poor)
- Bonus marks support

---

## 🧠 Concepts Used
- Variables (`let`, `const`)
- Conditional Statements (`if-else`)
- Ternary Operator (`? :`)
- Switch Case
- Operators (`+`, `>=`, `<`)

---

## 🔄 Program Flow

1. Input marks 🎯  
2. Add bonus marks ➕  
3. Check grade using conditions  
4. Decide pass/fail  
5. Print remarks
